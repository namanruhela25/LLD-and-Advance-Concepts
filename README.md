# LLD & Advanced Concepts

 Notes and implementations covering Low-Level Design fundamentals and advanced software engineering concepts.

## Contents

| Module | Description |
|---|---|
| `oop/` | Core OOP principles — encapsulation, inheritance, polymorphism, abstraction |
| `solid/` | SOLID design principles with examples |
| `patterns/` | Gang of Four design patterns (Creational, Structural, Behavioral) |
| `uml_diagrams/` | UML diagrams for class, sequence, and component modeling |
| `designs/` | End-to-end LLD case studies (e.g. Google Docs Document Editor) |

## Tech

- **Language:** Java

## Purpose

Study reference for system design interviews and building a strong foundation in object-oriented design.
